O que é o Claude Desktop

O Claude Desktop é o aplicativo nativo do Claude para Mac e Windows — uma versão instalável que roda fora do navegador, fica acessível direto da barra de tarefas ou do dock, e adiciona recursos que a versão web simplesmente não tem.

A capacidade de conversa é exatamente a mesma do Claude Chat no navegador — a diferença está em tudo que existe em volta da conversa: atalhos, extensões locais e automações mais profundas.

Como instalar — Mac e Windows

1

Acesse o site oficial

Vá até claude.com/download — esse é o único lugar seguro para baixar o aplicativo.

2

Escolha seu sistema

Selecione a versão para Mac ou para Windows, conforme o seu computador.

3

Execute o instalador

Abra o arquivo baixado e siga as instruções na tela até a instalação completar.

4

Abra o aplicativo e entre na conta

Procure o Claude na pasta de Aplicativos (Mac) ou no menu Iniciar (Windows), e faça login com a mesma conta que você já usa — e-mail ou Google.

Pronto: suas conversas, projetos, memória e preferências já aparecem sincronizados automaticamente, porque tudo é vinculado à mesma conta usada no navegador.

Cowork — o recurso mais poderoso do Desktop

Se existe um motivo para instalar o Claude Desktop além da conveniência, é o Cowork. Ele leva a capacidade de execução de tarefas do Claude Code para uma interface visual — sem precisar de linha de comando — permitindo entregar tarefas longas e com várias etapas para o Claude resolver enquanto você faz outra coisa.

Na prática, isso significa pedir coisas como organizar uma pasta inteira de arquivos, gerar uma planilha com fórmulas funcionando, sintetizar uma pesquisa a partir de várias fontes, ou produzir uma apresentação formatada — e o Claude entrega o resultado final, não só uma resposta de texto.

  • Acessa seus arquivos locais diretamente, sem você precisar fazer upload manual de cada um.
  • Roda em um ambiente isolado dentro do seu próprio computador — a leitura e escrita de arquivos fica limitada às pastas que você conectar.
  • Coordena várias frentes ao mesmo tempo, útil para projetos com múltiplas partes em paralelo.

Importante: o Cowork exige plano pago e o seu computador precisa estar ligado e acordado durante a execução — se a máquina entrar em suspensão, a tarefa para até você voltar a usá-la.

Primeiro uso, passo a passo

1

Explore o atalho de entrada rápida

Procure nas configurações do app qual é o atalho de teclado configurado, e teste abrir o Claude por cima de outro programa.

2

Conecte uma extensão

Vá até a seção de extensões nas configurações e conecte a primeira ferramenta que você usa no dia a dia.

3

Teste arrastar um arquivo

Pegue um documento qualquer e arraste direto para dentro da conversa — sem precisar do botão de anexo.

4

Se tiver plano pago, teste o Cowork

Dê uma tarefa real e com várias etapas, escolha uma pasta para trabalhar, e acompanhe o progresso.

Requisitos do sistema

De forma geral, o Claude Desktop funciona em versões recentes de macOS e Windows. Alguns recursos específicos — principalmente o Cowork — têm exigências mais altas, como determinadas edições do Windows ou processadores específicos da Apple.

Recomendação: como requisitos de sistema mudam com frequência e variam por recurso, confirme os requisitos mínimos atualizados direto na página oficial de download antes de instalar, especialmente se for usar o Cowork.

Segurança: de onde baixar (e de onde nunca baixar)

⚠ ATENÇÃO REDOBRADA

Baixe o Claude Desktop apenas em claude.com/download. Nunca instale a partir de outro site, anúncio, torrent ou loja de aplicativos não oficial. Erros comuns de quem está começando

1
Baixar de um site que não é o oficial. Sempre confira que a URL é claude.com antes de baixar qualquer instalador.
2
Nunca explorar as extensões. Quem só conversa com o Claude no Desktop está usando exatamente o que teria no navegador — o ganho real está nas extensões e na entrada rápida.
3
Esperar o Cowork funcionar com o computador suspenso. A tarefa precisa do computador ligado e acordado do início ao fim.
4
Confundir Desktop com Claude Code no terminal. O Desktop inclui acesso ao Claude Code de forma visual, mas é um produto diferente do uso via linha de comando — cada um serve a um perfil de usuário.
